How to Play (optional)

This is not part of the story — it's just a quick, spoiler-free guide to a couple of techniques this kind of game sometimes uses. If you've done this before, skip it. If not, this should help.

1. Viewing a page's source code

Every web page is built from text (HTML) that your browser turns into what you see. You can look at that raw text directly — sometimes there's more written there than what's shown on screen.

How: right-click anywhere on a page and choose View Page Source (or press Ctrl+U on Windows / Cmd+Option+U on Mac). A new tab opens showing the raw code. You can search it with Ctrl+F like any page.

Some of what's hidden there looks like this: <!-- a comment, like this one --> — text left behind that isn't meant to be seen normally.

2. Caesar ciphers

A Caesar cipher shifts every letter forward (or back) by the same fixed amount through the alphabet. A shifted by 3 becomes D, B becomes E, and so on. There are only 25 possible shifts — try them until real words appear.

3. Letter-number ciphers

Sometimes letters are represented by their position in the alphabet: A=1, B=2, C=3 ... Z=26. A string of numbers like 8-9 would spell HI.

4. Some things are shared, some are personal

A few discoveries in this world are shared — if someone else finds one of the personal records scattered through this site, a trace of it will appear for everyone, even if what's inside stays theirs to explore.

But your own path — what you unlock, what you're asked, what you find — is yours alone. No one can unlock a page for you, and no one can lock you out of one either.

That's it. Everything else, you find on your own.
